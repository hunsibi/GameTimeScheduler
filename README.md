# Game Time Scheduler (게임 시간 관리 스케줄러)

## 📌 프로젝트 소개
자녀의 올바른 게임 습관을 길러주고 게임 시간을 효과적으로 관리하기 위해 제작된 윈도우용 타이머 애플리케이션입니다. 
지정된 게임의 작동을 백그라운드에서 감지하고, 설정된 제한 시간이 경과하면 주기적인 알람을 주거나 자동으로 컴퓨터를 종료시킵니다.

## ✨ 주요 기능
- **자동 감지**: 등록된 특정 프로그램(게임)의 실행 프로세스를 백그라운드에서 스마트하게 모니터링
- **타이머 & 알림 시스템**: 제한 시간이 다가오면 윈도우 시스템 트레이 팝업(Floating Notification)으로 남은 시간 경고
- **자동 종료**: 사용 설정된 제한 시간에 도달하면 게임 프로세스를 종료하거나 PC를 자동 종료
- **시스템 트레이(System Tray)**: 작업 표시줄 하단 트레이 아이콘으로 숨어서 작동하며, 필요시 설정 창 호출 가능
- **비밀번호 잠금 보안**: 아이들이 임의로 타이머 제한 시간이나 기능 설정을 변경하지 못하도록 비밀번호를 통한 접근 제어

## 🚀 설치 및 실행 방법

### 분리된 단일 실행 파일 (EXE) 사용 환경
파이썬이 설치되지 않은 PC 환경이라도 `dist` 폴더 안에 패키징 된 `GameTimeScheduler.exe` 파일을 더블클릭하여 즉시 사용하실 수 있습니다.

### 소스코드로 직접 실행 (Python 환경)
파이썬(Python 3.x)이 설치되어 있다면 다음 과정으로 실행합니다:
```bash
# 1. 필수 라이브러리 설치
pip install -r requirements.txt

# 2. 프로그램 실행 (백그라운드 트레이로 들어갑니다)
python main.py
```

## 🛠 단일 EXE 빌드 방법
코드를 수정한 뒤 다시 `EXE` 파일 하나로 배포하고 싶으시다면 동봉된 `.bat` 스크립트를 사용하세요:
1. 폴더 내부의 `build.bat`을 실행합니다.
2. 스크립트가 `PyInstaller` 설치 및 패키징 과정을 자동 진행합니다.
3. 빌드가 완료되면 `dist/GameTimeScheduler.exe` 파일이 새롭게 생성됩니다.
